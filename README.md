# ESP Web Tool
A web app to flash your ESP32 or ESP8266 through your browser. Open-Source, free, and easy to use.

## Credits
- Based on [ESP Web Flasher](https://github.com/NabuCasa/esp-web-flasher)  
- Inspired by [esptool-js](https://github.com/espressif/esptool-js) 
- Originally build by [SpacehuhnTech](https://github.com/SpacehuhnTech/espwebtool)

## Installation

```
git clone https://github.com/WatskeBart/espwebtool
cd espwebtool
npm install
npm run
```
